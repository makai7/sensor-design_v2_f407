# 项目名称: STM32F407 智能云台感知系统
# 硬件平台: STM32F407VET6 (自定义板)
# 开发环境: STM32CubeIDE / GCC / HAL库

## 项目目标
开发一套基于 STM32F407 的固件，实现以下功能：
1. **云台控制**：通过舵机控制 PCB 或传感器支架进行方位扫描。
2. **环境感知**：
   - 使用超声波传感器测量前方物体距离。
   - 使用摄像头捕捉当前视角的图像数据。
3. **数据传输**：将采集到的距离信息、云台角度和图像状态通过串口发送给上位机。
